# Trabalho Machine Learning Unidade 3 
# VISÃO GERAL E OBJETIVO 
O objetivo deste trabalho é proporcionar aos alunos experiência prática na aplicação de modelos de Machine Learning (ML) para resolver um problema real. Os alunos deverão selecionar um conjunto de dados, definir métricas de avaliação, treinar e comparar modelos, e apresentar os resultados obtidos. 

O projeto deverá ser feito por **grupos de** **3 alunos.** 
# ESPECIFICAÇÕES DO PROJETO 
Os alunos devem desenvolver um projeto que inclua as seguintes etapas: 

1. Definição do Problema 
- Escolher um problema que possa ser resolvido com técnicas de Machine Learning (classificação, regressão, clustering, etc.). 
- Justificar a escolha do problema e sua relevância. 
2. ## Obtenção e Preparação dos Dados 
- Identificar e baixar um conjunto de dados público (ex: Kaggle, UCI Repository, Google Dataset Search) ou um conjunto de dados criado pelo próprio grupo. 
- Realizar pré-processamento (limpeza, normalização, codificação de variáveis categóricas, etc.). 
- Dividir os dados em conjuntos de treino, validação e teste. 
3. ## Seleção e Treinamento de Modelos 
- Escolher pelo menos **dois** algoritmos de Machine Learning (ex: Regressão Logística, Random Forest, SVM, Redes Neurais). 
- Treinar os modelos usando o conjunto de treino. 
- Ajustar hiperparâmetros usando validação cruzada ou busca em grade (Grid Search). 
4. ## Avaliação e Comparação de Modelos 
- Definir métricas de avaliação apropriadas (ex: Acurácia, Precisão, Recall, F1-Score, RMSE). 
- Comparar os modelos no conjunto de teste e discutir os resultados. 
# REQUISITOS TÉCNICOS 
- Utilizar Python (bibliotecas como scikit-learn, pandas, matplotlib). 
- Opcional: Uso de frameworks como TensorFlow/PyTorch para redes neurais. 
# ENTREGA 
Montar um repositório (Github, Google Colab) do projeto com os códigos e um readme apresentando todos os requisitos e passos para reproduzir os resultados alcançados.  

Preparar uma apresentação de slides contendo no mínimo os seguintes pontos: 

- Descrição do problema e dos dados. 
- Metodologia utilizada (pré-processamento, modelos testados). 
- Resultados e discussão (tabelas, gráficos, comparação de desempenho). 
- Conclusões e possíveis melhorias. 
- Link do repositório  

Somente um aluno do grupo deve enviar a apresentação (formato PDF), para o AVA, na seção entrega de trabalhos.
